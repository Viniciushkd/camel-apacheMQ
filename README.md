# Camel ApacheMQ
## Configuração
Instalar [apache-cxf-2.7.11](http://archive.apache.org/dist/cxf/2.7.11/)
### Windows
```
apache-cxf-2.7.11\bin
activemq start
```
### Linux 
```
sudo /opt/apache-activemq-5.15.4/bin/activemq console
~
sudo /opt/apache-activemq-5.15.4/bin/activemq start
```
